快捷键

*Alt+Z * (文件自动换行)

Ctrl + /** （单行注释）

*Shift + Alt + A *（多行注释） 若要取消单行或多行注释在按一次该快捷键即可

Ctrl + Shift + Enter （上方插入一行，鼠标光标在当前行的任意位置都可以直接换行到上一行）

*Ctrl + Enter *（下方插入一行，鼠标光标在当前行的任意位置都可以直接换行到下一行）

*Alt + Shift + F *（格式化代码，不需要全选中代码，直接格式化即可）

*Ctrl + Shift + F *（查找文件）

Ctrl + p右边会弹出所有快捷键目录

Ctrl + Shift + [ (代码折叠)

Ctrl + Shift + ] (代码展开)

*Shift + Alt + Up/Down * （向上/向下 复制行）

Ctrl + backspace （删除上一个单词）

F2 (对当前文件某一个单词全部重命名，上下文中的引用都会被重命名)

ctrl+F2 (找到当前文件所有某个单词，光标处于单词的末尾)

*多行转一行 *（选中，ctrl+shift+p 输入 join lines，点击设置可以设置快捷键alt+q）

去掉尾随空格

code改成中文

方法一：点击插件按钮搜索 Chinese

方法二：

1）打开vscode工具；

2）使用快捷键组合【Ctrl+Shift+p】，在搜索框中输入“configure display language”，点击确定后；

3）修改locale.json文件下的属性“locale”为“zh-CN”;

4）重启vscode工具；

或者 汉化中文（插件）

点击插件按钮搜索 Chinese, 在弹出的选项中选择第一个中文简体

然后右边会弹出如下图安装界面，接着点击 Install 安装

自动保存设置

File（文件）- Preferences（首选项）- Setting （设置）然后弹出下面界面，选择User（一般会默认选中该选项），

接着如下图选择afterdelay选项即可，下面有个数字1000表示1秒，这个可自行设置，表示多少秒以后自动保存

自动格式化代码

File（文件）- Preferences（首选项）- Setting （设置）

User（用户）- Text-Editor（文本编辑）- Formatng（格式化）

然后勾选下图红色框中的选项后重启Vscode即可

Vscode更换主题

File（文件）- Preferences（首选项）- Color-Theme （颜色主题）

然后会出现下图红色框的界面，此时按键盘上的上下键即可实时查看主题颜色，直接回车可选中对应主题

注：可以点击插件直接搜索 theme 下载其它非内置主题

  1. 打开vscode,选择文件–首选项–用户代码片段
  2. 选择javascript类型，打开javascript.json文件
  3. 按照操作，设置快捷输入

VSCODE快速生成标签

1、添加类、id、文本和属性

（1）输入 .box

（2）输入p#text

（3）类和id一起输入 .box #text

（4）元素和内容一块输出 p{你好}

（5）输出属性 a[href=#]

（6）给后代设置class1 ul>.calss

（7）给后代设置class2 table>.row>.col

2、同级和嵌套

（1）输入div+p(同级标签)

（2）输入ul>li*3>a（嵌套标签）

3、分组

可以通过嵌套和括号来快速生成一些代码块(.foo>h1)+(.bar>h2)

4、定义多个带属性的元素

（1）输入ul>li.item$*3，代码输出

（2）自增符$ div.class>div.class1_$*5

（3）自增 从某数开始 div.class>div.class1_$@3*4

（4）自减 从某数开始 div.class>div.class1_$@-3*4
