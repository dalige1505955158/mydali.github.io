vue-i18n的联动使用

需求描述：需要中英韩三种语言切换功能，第三方翻译工具有局限，1）翻译不准确，2）调用第三方接口有字数限，所以，最终决定翻译页面中静态文字，接口返回数据不翻译；

实现方案：elementUI提供的国际化多语言。翻译内容分俩部分：1）页面中自定义内容变量；2）elementUI自带插件（日历、表格翻页等）
