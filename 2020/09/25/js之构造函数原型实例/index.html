js之构造函数原型实例

发表于 2020-09-25 | 分类于 js

首先用构造函数（一会解释什么叫做构造函数）的方法创建一个对象

构造函数：现在M就是构造函数，因为他被new过了！–在js中，只要是被new过的函数都被成为构造函数。

实例对象：现在a1就叫做实例，因为他是被new出来的！–在js中，只要是被new出来的对象都叫做实例对象。也叫实例。

原型对象：现在M.prototype就是原型对象，构造函数函数被创建出来的时候，天生就带一个prototype属性，而这个属性里面的东西就叫原型对象。

\他们之间的关系就如同下图：**我们暂时只看绿色框中的内容

从图中可得知4种关系：

1、构造函数（例子中的M）new出来的东西就叫实例，或者叫实例对象

2、构造函数（例子中的M）的prototype属性就叫原型对象

3、原型对象（M.prototype）中的constructor其实就等于构造函数，可以打印一下 M.prototype.constructor===M //true

这样我们就知道构造函数和原型对象的关系了吧，就是一个闭环。

4、实例（例子中的a1）的_ _proto__属性其实就是把他new出来的那个构造函数的prototype

可以打印 a1._ proto_===M.prototype //true

接下来我们看图的另一半

一个被new出来的实例，通过本身的proto属性找到他的原型对象，然后再通过他的原型对象的proto找到他的原型对象，一直找，递归访问proto最终到头，并且值是null，这个过程从头到结束就叫原型链
