小程序开发之阿拉丁统计平台

发表于 2020-10-09 | 分类于 soft

关键步骤：

  • 注册/登录统计平台
  • 创建小程序
  • 接入SDK
  • 检测SDK
  • 集成成功

第一步：注册/登录 阿拉丁统计平台

阿拉丁统计平台注册可以通过 PC 访问：tj.aldwx.com，通过「微信扫码+手机号」的方式即可完成注册。注册流程大约只需要 19s。

)

如果您已经注册过阿拉丁统计平台，可以直接通过微信扫码的方式登录，方便快捷。

第二步： 创建阿拉丁小程序

登录阿拉丁统计平台后，会默认跳转到阿拉丁统计后台的主页。方便您了解所有小程序的数据变化。

)

我们通过右上角「创建应用」可以进入小程序的创建页面，开始我们小程序的创建工作。

)

创建小程序需要填写：小程序名称，选择小程序的类型（小程序、小游戏），选择小程序的分类（小程序对应的是小程序类型，小游戏对应的是小游戏类型）。

填写信息后，提交就可以获取AppKey，然后就开始SDK接入了。

第三步：接入阿拉丁统计SDK（重点）

接入SDK一直以来都是大家最关心的部分，也是接入阿拉丁统计最重要的一个步骤。所以这部分要认真看哦。提交信息后可以看到下面的截图：

)

注：阅读此部分的内容需要和开发人员一起，欢迎分享给自己本公司的小程序开发人员。

首先，这个部分的流程需要小程序的技术人员（一般是前端）来协助完成。在此过程中需要了解 阿拉丁Key、统计平台SDK、帮助文档以及数据检测。

阿拉丁Key（又称“AppKey”）是我们为大家创建小程序后生成的唯一凭证，开发者在集成SDK的时候需要将其填写到代码中（具体的操作见后文）。

其次，我们在接入的时候需要充分了解自己的小程序是否采用了框架（wepy、mpvue、Taro）或者插件（最好是和小程序的开发人员沟通确认），如果都没有的话就是标准的小程序类型。

不同的框架、插件接入流程会有细微的变化，但是阿拉丁Key和阿拉丁SDK是一样的。

接着，我们了解了阿拉丁Key和小程序类型后，我们可以下载SDK进行代码集成工作了。由于本文篇幅问题，本文的集成案例以「标准版小程序」演示。

实际操作

1、设置域名

登录微信公众平台(mp.weixin.qq.com)，在 开发→开发设置->服务器域名 找到「 request 合法域名」，点击「修改」，将log.aldwx.com 添加 request 合法域名中。

)

2、配置SDK

点击下载 微信小程序SDK，并将下载的文件解压，将解压后的文件 ald-stat.js 和 ald-stat-conf.js 复制到小程序根目录的 utils 文件夹中。

如果没有 utils 文件夹请在小程序项目的根目录下创建；

)

使用微信开发者工具打开小程序根目录 utils 文件夹下的 ald-stat-conf.js 文件，在浏览器登录阿拉丁小程序平台-管理-小程序列表-找到需要集成的小程序，复制对应小程序的 AppKey 到 exports.app_key = ""的双引号中间。

)

)

3、检测上报状态

在微信开发者工具中，查看控制台中的 Network 是否存在 d.html 开头的请求，点击后可看到 http://log.aldwx.com 的域名信息，且 status 状态为 200。

)

验证成功后，将所有代码提交至微信审核，审核通过后需发布版本。

以上流程全部配置完成后了，表示小程序已经成功接入了阿拉丁统计平台。

如果遇到问题可以咨询您的同事（前端技术人员）或者咨询阿拉丁统计平台客服（微信ID：alddandan)。

参数 string 动作的参数，不超过255个字符，支持特殊字符但不包括（英文逗号 , 英文冒号 : 点 .）
参数值 string 动作的参数值，不超过255个字符，支持特殊字符但不包括（英文逗号 , 英文冒号 : 点 .）
