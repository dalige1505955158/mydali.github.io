js之84个常用的JS代码片段

发表于 2021-03-15 | 分类于 js

*1、all * (important)

如果数组所有元素满足函数条件，则返回true。调用时，如果省略第二个参数，则默认传递布尔值。

2、allEqual (important)

判断数组中的元素是否都相等

3、approximatelyEqual

4、arrayToCSV

此段代码将没有逗号或双引号的元素转换成带有逗号分隔符的字符串即CSV格式识别的形式。

5、arrayToHtmlList

此段代码将数组元素转换成

  • 标记，并将此元素添加至给定的ID元素标记内。

    • 6、attempt

    此段代码执行一个函数，将剩余的参数传回函数当参数，返回相应的结果，并能捕获异常。

    7、average (important)

    此段代码返回两个或多个数的平均数。

    8、averageBy (important)

    求对象数组的属性之和，一个 map()函数和 reduce()函数结合的例子，此函数先通过 map() 函数将对象转换成数组，然后在调用reduce()函数进行累加，然后根据数组长度返回平均值。

    9、bifurcate

    此函数包含两个参数，类型都为数组，依据第二个参数的真假条件，将一个参数的数组进行分组，条件为真的放入第一个数组，其它的放入第二个数组。这里运用了Array.prototype.reduce() 和 Array.prototype.push() 相结合的形式。

    10、bifurcateBy (important)

    此段代码将数组按照指定的函数逻辑进行分组，满足函数条件的逻辑为真，放入第一个数组中，其它不满足的放入第二个数组 。

    这里运用了Array.prototype.reduce() 和 Array.prototype.push() 相结合的形式，基于函数过滤逻辑，通过 Array.prototype.push() 函数将其添加到数组中。

    11、bottomVisible (important)

    用于检测页面是否滚动到页面底部。

    12、byteSize

    此代码返回字符串的字节长度。这里用到了Blob对象，Blob（Binary Large Object）对象代表了一段二进制数据，提供了一系列操作接口。其他操作二进制数据的API（比如File对象），都是建立在Blob对象基础上的，继承了它的属性和方法。生成Blob对象有两种方法：一种是使用Blob构造函数，另一种是对现有的Blob对象使用slice方法切出一部分。

    13、capitalize (important)

    将字符串的首字母转成大写,这里主要运用到了ES6的展开语法在数组中的运用。

    14、capitalizeEveryWord (important)

    将一个句子中每个单词首字母转换成大写字母，这里中要运用了正则表达式进行替换。

    15、castArray (important)

    此段代码将非数值的值转换成数组对象。

    16、compact

    将数组中移除值为 false 的内容。

    17、countOccurrences (important)

    统计数组中某个值出现的次数。

    18、Create Directory

    此代码段使用 existSync() 检查目录是否存在，然后使用 mkdirSync() 创建目录（如果不存在）。

    19、currentURL

    返回当前访问的 URL 地址。

    20、dayOfYear (important)

    返回当前是今年的第几天

    21、decapitalize (important)

    将字符串的首字母转换成小写字母。

    22、deepFlatten (important)

    通过递归的形式，将多维数组展平成一维数组。

    23、default (important)

    去重对象的属性，如果对象中含有重复的属性，以前面的为准。

    24、defer

    延迟函数的调用，即异步调用函数。

    25、degreesToRads

    此段代码将标准的度数，转换成弧度。

    26、difference (important)

    此段代码查找两个给定数组的差异，查找出前者数组在后者数组中不存在元素。

    27、differenceBy (important)

    数组对象，通过给定的函数来处理需要对比差异的数组，查找出前者数组在后者数组中不存在元素。

    28、differenceWith (important)

    此段代码按照给定函数逻辑筛选需要对比差异的数组，查找出前者数组在后者数组中不存在元素。

    29、digitize

    将输入的数字拆分成单个数字组成的数组。

    30、distance

    计算两点之间的距离

    31、drop (important)

    此段代码将给定的数组从左边开始删除 n 个元素

    32、dropRight

    此段代码将给定的数组从右边开始删除 n 个元素

    33、dropRightWhile

    此段代码将给定的数组按照给定的函数条件从右开始删除，直到当前元素满足函数条件为True时，停止删除，并返回数组剩余元素。

    34、dropWhile (important)

    按照给的的函数条件筛选数组，不满足函数条件的将从数组中移除。

    35、elementContains

    接收两个DOM元素对象参数，判断后者是否是前者的子元素。

    36、filterNonUnique (important)

    移除数组中重复的元素

    37、findKey (important)

    按照给定的函数条件，查找第一个满足条件对象的键值。

    38、findLast (important)

    按照给定的函数条件筛选数组，将最后一个满足条件的元素进行删除。

    39、flatten (important)

    按照指定数组的深度，将嵌套数组进行展平。

    40、forEachRight

    按照给定的函数条件，从数组的右边往左依次进行执行。

    41、forOwn

    此段代码按照给定的函数条件，支持三个参数作为输入（值、键、对象本身），进行迭代对象。

    42、functionName

    此段代码输出函数的名称。

    43、getColonTimeFromDate

    此段代码从Date对象里获取当前时间。

    44、getDaysDiffBetweenDates (important)

    此段代码返回两个日期之间相差多少天。

    45、getStyle

    此代码返回DOM元素节点对应的属性值。

    46、getType (important)

    此段代码的主要功能就是返回数据的类型。

    47、hasClass

    此段代码返回DOM元素是否包含指定的Class样式。

    48、head

    此段代码输出数组的第一个元素。

    49、hide

    此段代码隐藏指定的DOM元素。

    50、httpsRedirect

    此段代码的功能就是将http网址重定向https网址。

    51、indexOfAll (important)

    此代码可以返回数组中某个值对应的所有索引值，如果不包含该值，则返回一个空数组。

    52、initial (important)

    此段代码返回数组中除最后一个元素的所有元素。

    53、insertAfter

    此段代码的功能主要是在给定的DOM节点后插入新的节点内容

    54、insertBefore

    此段代码的功能主要是在给定的DOM节点前插入新的节点内容

    55、intersection (important)

    此段代码返回两个数组元素之间的交集。

    56、intersectionBy (important)

    按照给定的函数处理需要对比的数组元素，然后根据处理后的数组，找出交集，最后从第一个数组中将对应的元素输出。

    57、intersectionBy

    按照给定的函数对比两个数组的差异，然后找出交集，最后从第一个数组中将对应的元素输出。

    58、is

    此段代码用于判断数据是否为指定的数据类型，如果是则返回true。

    59、isAfterDate (important)

    接收两个日期类型的参数，判断前者的日期是否晚于后者的日期。

    60、isAnagram

    用于检测两个单词是否相似。

    61、isArrayLike

    此段代码用于检测对象是否为类数组对象,是否可迭代。

    62、isBeforeDate

    接收两个日期类型的参数，判断前者的日期是否早于后者的日期。

    63、isBoolean

    此段代码用于检查参数是否为布尔类型。

    64、getColonTimeFromDate

    用于判断程序运行环境是否在浏览器，这有助于避免在node环境运行前端模块时出错。

    65、isBrowserTabFocused

    用于判断当前页面是否处于活动状态（显示状态）。

    66、isLowerCase

    用于判断当前字符串是否都为小写。

    67、isNil

    用于判断当前变量的值是否为 null 或 undefined 类型。

    68、isNull

    用于判断当前变量的值是否为 null 类型。

    69、isNumber

    用于检查当前的值是否为数字类型。

    70、isObject

    用于判断参数的值是否是对象，这里运用了Object 构造函数创建一个对象包装器，如果是对象类型，将会原值返回。

    71、isObjectLike

    用于检查参数的值是否为null以及类型是否为对象。

    72、isPlainObject

    此代码段检查参数的值是否是由Object构造函数创建的对象。

    73、isPromiseLike

    用于检查当前的对象是否类似Promise函数。

    74、isSameDate (important)

    用于判断给定的两个日期是否是同一天。

    75、isString

    用于检查当前的值是否为字符串类型。

    76、isSymbol

    用于判断参数的值是否是 Symbol 类型。

    77、isUndefined

    用于判断参数的类型是否是 Undefined 类型。

    78、isUpperCase

    用于判断当前字符串的字母是否都为大写。

    79、isValidJSON

    用于判断给定的字符串是否是 JSON 字符串。

    80、last

    此函数功能返回数组的最后一个元素。

    81、matches

    此函数功能用于比较两个对象，以确定第一个对象是否包含与第二个对象相同的属性与值。

    82、maxDate

    此代码段查找日期数组中最大的日期进行输出。

    83、maxN

    此段代码输出数组中前 n 位最大的数。

    84、minDate

    此代码段查找日期数组中最早的日期进行输出。
